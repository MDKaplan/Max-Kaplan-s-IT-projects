# Max-Kaplan-s-IT-projects
IT projects I have done so far. Includes coding in C++, python, using VMware, cloud creation, ect.





ticketscreenshots - Cloud based IT Service Management System
  Deployed a cloud hosted ITSM ticketing platform on a Linux virtual machine using OCI. Installed and implemented a LAMP stack, configured database backend, enabled secure admin access through SSH.

ADinVM - Active Directory Home Lab
  This project demonstrates the setup of an AD environment using Windows Server 2022 VM in Hyper-V. The server was initially configured by installing AD DS and DNS, then promoting the server to create a new domain. This includes user account creation and organizational unit management to simulate real world domain     structure.

connectfour6.exe, connectfour7.c, connectfour7.h - Connect 4 in C
    Developed command line implementation of Connect 4 in C. Includes statement management, player turns, win/ loss calculation, move validation.

cipher.py - Hill Cipher in Python
  Developed a Python based cryptography application implementing the Hill cipher algorithm using matrix based transformers.

pa01.py - Checksum Validator in Python
  Built a Python checksum program to verify data integrity by calculating and validating checksum across input data. Simulated real world integrity checking mechanisms used in systems and networking contexts.
